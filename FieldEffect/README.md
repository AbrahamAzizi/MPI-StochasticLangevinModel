
Project Overview

This project employs a stochastic Langevin equation incorporating both Brownian and Néel relaxation mechanisms to model the magnetic dynamics of various magnetic nanoparticle (MNP) tracers under different excitation fields.

To evaluate system behavior and performance, the following were analyzed:

Time-domain magnetization curves (M–t)
Magnetic hysteresis loops (M–H)
Point Spread Functions (PSFs)
Higher-order harmonics
Signal-to-noise ratio (SNR) of the third harmonic

These analyses help assess how excitation field parameters influence Magnetic Particle Imaging (MPI) performance.

Evaluation Metrics
Full Width at Half Maximum (FWHM): used to quantify imaging resolution
Signal-to-Noise Ratio (SNR): used to evaluate signal quality
